## 🧠 COMPONENT DESCRIPTIONS

### 1. NET Layer (`nx_secure_server.py`)
- **Purpose**: Handles external, remote intelligence streams.
- **Security**: Encrypted via TLS 1.3+.
- **Isolation**: Completely separated from local logic to prevent external interference.
- **Protocol**: IQ4_NL over TCP/WebSocket.

### 2. LAN Layer (`user_config_logic.py`)
- **Purpose**: Direct injection of user-defined configuration logic.
- **Speed**: Zero network overhead, instant execution.
- **Privacy**: Never exposed to the network.
- **Usage**: Set elliptical angles, seismic thresholds, quantum parameters.

### 3. CORTEX A-72 Bridge (`cortex_a72_bridge.py`)
- **Purpose**: The central neural interface routing all traffic.
- **Functions**:
  - **Synaptic Transform**: Converts RA/Dec to Elliptical X/Y.

  - **Priority Routing**:
    - `CRITICAL` → Logic Layer Only.
    - `HIGH` → Logic Layer + System.
    - `NORMAL` → System Only.
    - `NOISE` → Discarded.

### 4. Logic Layer
- **Role**: Receives high-priority "IQ Direct" packets.
- **Action**: Triggers alerts, updates quantum state, executes user scripts.

### 5. System Rest (Visualizer/Storage)
- **Role**: Receives all non-noise data for display and archiving.
- **Output**: Pictography glyphs (⚡🌌✨·), XY curves, star maps.
